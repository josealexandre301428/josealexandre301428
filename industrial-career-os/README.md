# Industrial Career OS

Sistema pessoal de carreira, documentação técnica e portfólio industrial de **José Alexandre Pinto Filho**.

## Objetivo

Transformar experiências reais em automação, instrumentação, elétrica industrial e dados em ativos profissionais reutilizáveis:

- Estudos de caso técnicos;
- Biblioteca técnica industrial;
- Projetos Python, SQL e Power BI;
- Dashboards de manutenção;
- Materiais para currículo, entrevistas e LinkedIn;
- Base para oportunidades nacionais e internacionais.

## Áreas de foco

- Automação Industrial
- Instrumentação
- Elétrica de Força e Controle
- Mineração
- Comissionamento
- Dados Industriais
- Python, SQL e Power BI

## Regras do projeto

1. Não expor informações confidenciais de empresas ou clientes.
2. Remover nomes internos, documentos proprietários e dados sensíveis.
3. Transformar atividades reais em aprendizado técnico reutilizável.
4. Todo projeto deve ter contexto, problema, solução, tecnologias e aprendizados.
5. Priorizar clareza para recrutadores e gestores técnicos.
